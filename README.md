## 🎮 Scripts

### JujutsuRandomizer.lua

A comprehensive UI script for the Jujutsu Kaisen Roblox game featuring:

#### 🔧 Features

**Main Features:**
- **Auto Fight System**: Automatically fights enemies with smart targeting
- **Auto Yen Collection**: Automatically collects yen from designated locations
- **Smart Targeting**: Prioritizes closest enemies within range

**Visual Features:**
- **ESP (Extra Sensory Perception)**: Player outlines and health bars
- **Tracers**: Lines pointing to player positions
- **Aim Lock**: Automatic camera aiming at targets
- **Health Prediction**: Predicts enemy health changes

**UI Features:**
- **Draggable Interface**: Move the UI anywhere on screen
- **Minimizable**: Minimize to title bar only
- **Theme System**: Built-in color themes (Dark, Blue, Red)
- **Color Wheel**: Custom color selection
- **UI Blur Toggle**: Toggle background blur effect

#### 🎨 UI Controls

- **K Key**: Toggle main UI on/off
- **Drag**: Click and drag the title bar to move
- **Minimize**: Click the `_` button to minimize
- **Close**: Click the `X` button to hide UI

#### 🔑 Key System

The script includes a key verification system:
- Visit the provided link to obtain a key
- Enter the key in the key input field
- The key is saved locally for future use

## 🚀 Installation

1. Download the desired script file
2. Open your Roblox game
3. Use your preferred executor to load the script
4. The UI will appear on screen
